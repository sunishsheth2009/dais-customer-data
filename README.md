# DAIS demo — Customer Data

Per-customer notes used by the DAIS AI-Governance breakout demo. 30 customer
files under `customers/`, plus a `pii/` directory used in the policy demo.

Two consumers in the demo:
- **Governed app** uses `demos.dais.github-mcp-dais` (4 service policies, curated tool surface).
- **Ungoverned app** uses `demos.dais.github-mcp-ungoverned` (no policies, all tools).
